# Manual do Usuário

## Como utilizar o sistema

1. Acesse a página inicial.
2. Faça o cadastro ou login.
3. Navegue pelos produtos.
4. Utilize a busca ou os filtros.
5. Adicione produtos ao carrinho.
6. Revise o carrinho.
7. Finalize a compra.

## Administrador

O administrador pode:

- Cadastrar produtos.
- Editar produtos.
- Excluir produtos.
- Gerenciar pedidos.
- Controlar o estoque.
